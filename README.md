# Message Bookmarks - SillyTavern Extension

채팅 메시지에 북마크, 형광펜(하이라이트), 메모를 추가할 수 있는 SillyTavern 확장프로그램입니다.

## 기능

### 📑 북마크
- 원하는 메시지에 북마크를 추가하여 쉽게 찾을 수 있습니다
- 각 북마크에 메모를 남길 수 있습니다
- 북마크 패널에서 모든 북마크를 한눈에 확인할 수 있습니다

### 🖍️ 하이라이트 (형광펜)
- 메시지 내 텍스트를 선택하여 5가지 색상으로 하이라이트할 수 있습니다
  - 노랑, 초록, 파랑, 분홍, 주황
- 각 하이라이트에 메모를 추가할 수 있습니다
- 하이라이트를 클릭하여 메모 확인/편집/삭제가 가능합니다

### 📝 메모
- 북마크와 하이라이트에 각각 메모를 남길 수 있습니다
- 중요한 내용이나 나중에 참고할 내용을 기록하세요

## 사용 방법

### 하이라이트 추가
1. 채팅 메시지에서 하이라이트하고 싶은 텍스트를 드래그하여 선택합니다
2. 마우스 우클릭을 합니다
3. 원하는 색상을 선택합니다
4. (선택사항) 메모를 입력합니다

### 북마크 추가
1. 메시지 위에 마우스를 올리면 나타나는 북마크 버튼(🔖)을 클릭합니다
2. 메모를 입력하고 확인을 누릅니다

또는
1. 텍스트를 선택한 후 우클릭 메뉴에서 "메시지 북마크"를 선택합니다

### 북마크 패널
1. 설정 > Extensions > Message Bookmarks에서 "북마크 패널 열기" 버튼을 클릭합니다
2. 모든 북마크 목록을 확인할 수 있습니다
3. 클릭하면 해당 메시지로 이동합니다

## 설치

1. `data/default-user/extensions/` 폴더에 `Message-Bookmarks` 폴더를 생성합니다
2. 이 확장프로그램의 파일들을 해당 폴더에 복사합니다
3. SillyTavern을 재시작하거나 새로고침합니다

## 파일 구조

```
Message-Bookmarks/
├── manifest.json    # 확장프로그램 메타데이터
├── index.js         # 메인 로직
├── style.css        # 스타일
└── README.md        # 이 문서
```

## 데이터 저장

- 북마크와 하이라이트 데이터는 채팅 메타데이터에 저장됩니다
- 각 채팅별로 별도로 관리됩니다
- 채팅을 삭제하면 해당 채팅의 북마크/하이라이트도 함께 삭제됩니다
